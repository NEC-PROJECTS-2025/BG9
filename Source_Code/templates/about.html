About This Project

The **Explainable Fetal Ultrasound Classification** project aims to classify fetal ultrasound image planes using deep learning models like Convolutional Neural Networks (CNN) and Multi-Layer Perceptron (MLP). Our goal is to apply **Explainable AI (XAI)** techniques to improve the transparency and interpretability of model predictions, particularly in the medical field where understanding model decisions is critical.

This system uses a combination of **image classification** techniques and **medical imaging** to categorize ultrasound images into six classes:

  • Fetal Abdomen
  • Fetal Brain
  • Fetal Femur
  • Fetal Thorax
  • Maternal Cervix
  • Other

The importance of this project lies in the integration of **deep learning** models with **explainability** methods, allowing clinicians to not only obtain accurate results but also gain insights into the reasoning behind these predictions. This level of transparency can be used to assist doctors in making more informed decisions.

Key Features:

  • Utilization of CNNs for image classification and MLP models for further analysis.
  • Application of **LIME** and **SHAP** for explainability and transparency in model predictions.
  • High accuracy in fetal ultrasound image classification to assist medical professionals.
  • Interactive interface for easy integration with healthcare applications.

The project also aims to provide a **user-friendly interface** for clinicians to easily upload ultrasound images and receive instant predictions with visual explanations on how the model arrived at its decision.
